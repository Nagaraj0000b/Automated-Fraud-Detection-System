Q1. a) Prepare a Test Plan for your project. The plan should include:
● Objective of testing
● Scope (modules/features to be tested)
● Types of testing to be performed (e.g., unit, integration, system)
● Tools (if any)
● Entry and Exit criteria
b) Design at least 8 test cases for any one major module of your project.
Each test case should include:
● Test Case ID
● Test Scenario / Description
● Input Data
● Expected Output
● Actual Output
● Status (Pass/Fail)

[Marks = 5+5]
Q2. a) Execute the test cases designed above and document the results
with evidence (screenshots/logs).
b) Identify and analyze at least 3 defects (bugs) found during testing. For
each defect, provide:
● Bug ID
● Description of the issue
● Steps to reproduce
● Expected vs Actual Result
● Severity level (Low/Medium/High)
● Suggested fix
