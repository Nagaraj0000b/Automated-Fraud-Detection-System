White Box Testing (also known as Glass Box Testing) is a testing
technique where the tester has full knowledge of the internal structure,
code, and logic of the application. Black Box Testing (also known as
Functional Testing) is a software testing method where the tester
evaluates the software without knowing its internal code or structure.
Write different test cases for performing White Box Testing and Black
Box Testing. Also, perform the testing using these test cases.
